Ideas here are a mix of ideas, they: might not be possible, might be hard to do, might be undesirable

User needs

Not a user need - 'as a user i want search results' (not something they would say, contains a solution, etc). But as part of the journey we are asking the user has to pick the right title record that will contain the fact that answers their question

Note: capturing, wrtitng and sharing user needs, and how to write user stories

We'll list here, more we know about how users might find the right title record, and existing problems they have in similar service (like FaP), and current FPI and things we've learnt from the lab, and other forms of feedback
