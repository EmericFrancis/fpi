Address based search examples/ideas/issues

Results

FPI currently a result per title1 (tenure), showing the A1 Property description - and the property description is the link.

Moving to a results per address, with all the tenures for that address grouped to that result, and displaying the postal address not the property description

1: So is Find a Property - in both services people pick/buy the wrong record becasue of this

A result per title (tenure), showing the A1 address, (current FPI), means for the user:

  • the address does not match their mental model of postcodes and addresses
  • more results for the user to deal with
  • people pick / buy the wrong document
  • less properties/titles can be found - hits the license limit more frequently.

A result per postal address, showing the postal address, means for the user:

  • matches the postcode model
  • help people understand that it's not one title record per property - that they will have to make a further choice
  • more properties and titles can be found - in total, and - found first time
